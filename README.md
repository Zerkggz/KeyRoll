##KeyRoll

KeyRoll automatically tracks party member keystones and allows you to "roll the dice" to pick which key your group should do.

Designed for World of Warcraft: Midnight and later.

##Features

* Automatic keystone tracking
* Parses and stores keystones shared via addons:
	* BigWigs / BigWigsKey
	* DBM / DBM-Key
	* MDT
	* AngryKeystones
	* LibKS
* Party keystone rolls
	* /keyroll rolls on a random keystone in the party.
	* Adds flavor text to simulate the roll experience.
	* Announces results to the party.
* Party keystone listing
	* /keyroll list displays all cached party keystones along with dungeon names and levels.
	* Automatically uses proper dungeon names in non-debug mode.
* Manual capture of keystones
	* /keyroll capture requests all party members to broadcast their keystones.
* Smart cache that updates automatically
* Automatically removes keys when players leave the party
